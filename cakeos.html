CakeOS

Download

Based on Niivu's beautiful CakeOS Windows Theme, this suite of skins attempts to emulate it to the tee.

Requirements:

  • ImageMagick
  • WebNowPlaying Companion (Only required for WebNowPlaying)
  • Spicetify (Set player to "Web" to useâ€” not Spotify)

Settings:

  • Theme (Boolean) - Changes the current color scheme
  • Edges (Trilean) - Changes the current edge scheme between Borderless, Bordered, and Rounded
  • Player - Changes the media player and/or plugin
  • Unit (Boolean) - Changes the weather unit between Fahrenheit and Celsius
  • Location - Changes the weather location to the town or city entered in the box

Hidden Variables:

  • SkinH (in every config) - Changes the height of an individual config
  • SkinW (in every config) - Changes the width of an individual config
