Drift

Download

Inspired by a dribbble post, this skin is all about motion. Each image slides up to reveal something hidden underneath, whether it be a shortcut or a set of controls.

Requirements:

  • ImageMagick
  • WebNowPlaying Companion (Only required for WebNowPlaying)
  • Spicetify (Set player to "Web" to use— not Spotify)

Settings:

  • ColorBG - Changes the background color
  • ColorFG - Changes the foreground color
  • Scale - Changes the scale of all configs
  • Padding - Changes the padding of each config— mostly affects images
  • Anim Speed - Multiplies the speed of each animation by this number
  • Blur Intensity - Determines the intensity of each image's blur when hovered over
  • Font - Changes the font for every config
  • Images Tab - Allows you to set custom images by changing the path to them

Hidden Variables:

  • SkinH (in every config) - Changes the height of an individual config
  • SkinW (in every config) - Changes the width of an individual config
  • SkinY (in every config) - Not recommended to change— Changes the starting height of the config's image

Known Bugs:

  • None
