Peroxide

Download

A worthy successor to Proximity, Peroxide employs even more of MagickMeter's features, including the use of ImageMagick's shadows.

Settings:

  • Color - Changes the foreground color
  • Font - Changes the font
  • Scale - Changes the size of the entire skin
  • Shadow Radius - Changes the size of the shadows
  • Shadow Alpha - Changes the opacity of the shadows
  • Corner Radius - Changes the size of the corners
  • Padding - Changes the padding inside of the skin
  • Alignment (Trilean) - Changes the alignment of all elements in the skin
  • Progress Background (Boolean) - Determines whether to show the progress bar in the background
  • Invert Progress Color - Changes the progress bar's color between black and white
  • Progress Alpha - Changes opacity of the progress bar
  • Background Brightness - Changes the brightness of the background cover
  • Cover (Boolean) - Determines whether to show the foreground cover
  • Cover Size - Changes the size of the foreground cover

Hidden Variables:

  • SkinH (in Variables.inc) - Changes the skins's height
  • SkinW (in Variables.inc) - Changes the skins's width
