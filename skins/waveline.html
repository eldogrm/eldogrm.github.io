This customizable visualizer is made up of only one shape meter to conserve cpu usage.

Settings:

  • Bar Count - Changes the amount of bars
  • Bar Spacing - Changes the amount of spacing in between each bar
  • Bar Width - Changes the width of each individual bar
  • Bar Height - Changes the max height of the bars
  • Bar Stroke W - Changes the stroke width of the bars
  • Gradient (Boolean) - WARNING: SPIKES CPU USAGEâ€” Determines whether or not to overlay a gradient onto each individual bar
  • Fill Color 1 - Determines the top color of the gradient or the main color
  • Fill Color 2 - Determines the bottom color of the gradient
  • Stroke Color 1 - Determines the top stroke color of the gradient or the main stroke color
  • Stroke Color 2 - Determines the bottom stroke color of the gradient
