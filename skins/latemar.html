Requirements:

  • WebNowPlaying Companion (Only required for WebNowPlaying)
  • Spicetify (Set player to "Web" to useâ€” not Spotify)

Settings:

  • ColorBG1 - Changes the background color of the notes skin
  • ColorBG2 - Changes the background color of floating elements
  • ColorACC - Changes the accent color
  • ColorFG - Changes the foreground color
  • SidebarAlpha - Changes the background transparency of the clock and note configs
  • FloatingAlpha - Changes the background transparency all other configs
  • Font - Changes the font
  • Scale - Changes the scale of all configs
  • SidebarPad - Changes the padding of the clock and note configs
  • FloatingPad - Changes the padding of all other configs
  • LineBreak - Changes the markdown required for a line break
  • BPoint - Changes the markdown required for a bullet point
  • Underline - Changes the markdown required to underline text
  • Bold - Changes the markdown required to bold text
  • Strikethrough - Changes the markdown required to strikethrough text

Hidden Variables:

  • SkinH (in graph configs, time, and notes) - Changes the skins's height
  • SkinW (in graph configs, time, and notes) - Changes the skins's width
