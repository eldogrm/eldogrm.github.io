Bouquet Suite II

Download

Based off of another Windows Theme courtesy of Niivu, this suite attempts to replicate a flowery yet sharp unix rice.

Requirements:

  • WebNowPlaying Companion (Only required for WebNowPlaying)
  • Spicetify (Set player to "Web" to use— not Spotify)

Settings:

  • Foreground Color - Changes the foreground color
  • Background Color - Changes the background color
  • Accent Color 1 - Changes the primary accent color
  • Accent Color 2 - Changes the secondary accent color
  • Scale - Changes the scale of every config
  • Border (Boolean) - Determines whether or not to display the border around each config
  • Shadow Radius - The radius of the shadow around each config
  • Background Alpha - The opacity of each config's background
  • Blur (Boolean) - Determines whether to display blur behind each config
  • Font - Determines which font to use
  • Font Size - Determines the base size of the font
  • Antialiasing - Determines whether or not to antialias text— keep this off when using bitmap fonts
  • 24 Hour Time (Boolean) - Determines whether or not to use 24 hour time
  • 24 Hour Time (Boolean) - Determines whether or not to display the AM/PM indicator
  • Sectioned Background (Boolean) - Changes the layout of the screenfetch (Power config)
  • Alignment (Trilean) - Changes the layout of the media player config
  • Album Text (Boolean) - Determines whether to show the currently playing album title
  • Cover Padding (Boolean) - Determines whether to apply the Border padding the the cover config
  • Progress Bar Type (Boolean) - Determines whether to fill the progress bar
  • Visualizer (Boolean) - Determines whether to show a visualizer or song position/duration underneath the controls
  • Visualizer Bar Width - Changes the width of each individual bar
  • Visualizer Smoothing (Boolean) - Determines whether to smooth the visualizer or lock it to a height that is a multiple of Visualizer Bar Width

Hidden Variables:

  • SkinH (in every config) - Changes the height of an individual config
  • SkinW (in every config) - Changes the width of an individual config

Known Bugs:

  • None
