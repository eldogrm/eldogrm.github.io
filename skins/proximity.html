One of the first skins to use Khanhas' MagickMeter plugin, this skin is able to easily blur and manipulate album art.

Requirements:

  • ImageMagick
  • WebNowPlaying Companion (Only required for WebNowPlaying)
  • Spicetify (Set player to "Web" to useâ€” not Spotify)

Variables:

  • Plugin - Changes the current plugin fetching music information
  • Player - Changes the current player for the NowPlaying plugin
  • Cover (Boolean) - Determines whether to show the cover
  • Full (Boolean) - Determines whether to show the uncropped cover
  • Ctrls (Boolean) - Determines whether to show the music controls
  • Color - Changes the foreground color
  • Round (Boolean) - Determines whether to round the skin's corners
  • Scale - Changes the skin's overall size
  • Font - Changes the skin's font
  • Vis (Boolean) - Determines whether to show the visualizer
  • BarH - Changes the max height of the bars
  • BarW - Changes the width of the bars
